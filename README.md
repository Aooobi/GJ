# 花火镇：献祭

## 🎮 游戏简介

一名病危之人转生到神秘的"火花镇"，通过战斗与抉择，解开火焰花背后的秘密。

## 🚀 快速开始

### 环境准备
- **Unity编辑器**：2022.3.62f3c1 LTS（必须完全一致）
- **Git**：>= 2.45.0

### 获取项目
```bash
git clone https://github.com/Aooobi/GJ.git
cd GJ
```

### 配置Git（避免行尾符冲突）
```bash
# macOS/Linux
git config core.autocrlf input

# Windows
git config core.autocrlf true
```

## 📁 项目结构

```
Assets/
├── Scripts/         # 游戏逻辑脚本
├── Art/             # 美术资源
├── Scenes/          # 游戏场景
└── Prefabs/         # 预制体
```

## 🎨 美术需求

### 瓦片地图
3种地形 × 2种风格 = 6套瓦片图集：
1. 草地（村庄区域）
2. 沼泽（第一层战斗区）
3. 森林（第二层战斗区）
   - 炽热风格：红、黄、橙色
   - 自然风格：绿、蓝、薄荷色

### 角色设计
- **主角**：病危转生者形象
- **NPC**：7位村民各具特色
- **怪物**：炽热蜗牛、火花蘑菇、灰烬树人、业火之花BOSS

### UI设计
- 统一风格的按钮和对话框
- 简约的背包和技能界面
