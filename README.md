# 花火镇：献祭

## 🎮 游戏简介

**一句话描述**：一名病危之人转生到神秘的"火花镇"，通过战斗与抉择，解开火焰花背后的秘密。

**核心玩法**：2D横板动作RPG + 道德抉择系统
- **探索**：三个战斗区域，每个区域有"炽热"和"自然"两种状态
- **战斗**：击败怪物获得花火，可学习怪物技能
- **献祭**：用花瓣或花火献祭，获得各种增益效果
- **睡眠**：通过睡眠补充资源，推进时间

## 🌟 游戏特色

### 道德抉择系统
- 解放战斗区域会永久减少村民的祈祷花瓣
- NPC态度随玩家行动而变化
- 隐藏剧情线：种植人"初"的私心

### 动态世界变化
- 清理怪物后区域从"炽热"变为"自然"状态
- 村庄命运随玩家选择而变化
- 多结局设定

## 🎨 美术需求

### 瓦片地图
3种地形 × 2种风格 = 6套瓦片图集：
1. 草地（村庄区域）
2. 沼泽（第一层战斗区）
3. 森林（第二层战斗区）
   - 炽热风格：红、黄、橙色
   - 自然风格：绿、蓝、薄荷色

### 角色设计
- **主角**：病危转生者形象
- **NPC**：7位村民各具特色

### UI设计
